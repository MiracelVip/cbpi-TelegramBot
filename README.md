# cbpi-TelegramBot
A bot for craftbeerpi3

## Installation
.. code:: shell

    $ sudo apt-get install python-pip
    
.. code:: shell

    $ pip install python-telegram-bot --upgrade
    
.. code:: shell

    $ cd craftbeerpi3/modules/plugins
    
.. code:: shell

    $ git clone https://github.com/MiracelVip/cbpi-TelegramBot
